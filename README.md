# liber-be

Backend for liber-fe with MongoDB, JWT auth, and bcrypt.

## Setup

1. Create a .env file based on .env.example.
2. Install dependencies.
3. Run the server.

## Scripts

- start: Run the server
- dev: Run the server with nodemon

## Endpoints

- POST /api/auth/register
- POST /api/auth/login
- GET /api/health

## Login Page

Open /login.html in the browser or navigate to http://localhost:5000/login.html after starting the server.
